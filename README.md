# PDF to Audio Converter

A simple Python script that extracts text from a PDF file and converts it into an audio file (MP3) using text-to-speech. This project is useful for accessibility, studying, or listening to documents hands-free.

# Features

Reads text from multi-page PDF files

Cleans and formats extracted text

Converts text into an MP3 audio file

Uses offline text-to-speech (no API keys required)

# Requirements

Python 3.8+

Required libraries:

PyPDF2

pyttsx3

# Install dependencies:

``` pip install PyPDF2 pyttsx3 ```

# Usage

Place the PDF you want to convert in the project directory

Update the file path in the script if necessary

# Run the script:

``` python main.py ```


The generated audio file (story.mp3) will appear in the same directory
